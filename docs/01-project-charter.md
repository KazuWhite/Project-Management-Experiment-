# 01 项目启动说明

## 1. 项目名称

校园活动报名与签到管理系统

## 2. 项目目标

- 本系统旨在解决校园活动组织过程中信息分散、报名混乱、 签到效率低下等问题，具体目标如下：  **-** **\*\***支持活动发布：**\*\*** 管理员可创建并发布活动基本信息，   包括活动名称、时间、地点、人数上限等字段 **-** **\*\***支持学生报名：**\*\*** 学生可浏览活动列表，选择感兴趣的活动完成在线报名，   系统自动记录报名状态 **-** **\*\***支持签到与统计：**\*\*** 活动现场支持签到操作，   系统自动汇总参与人数并生成统计结果支持活动发布
- 支持学生报名
- 支持签到与统计
